get /customers/search

Search for customers by first name, last name or address

  • Request
  • Response

Query Parameters

  • firstName: (string)

    First name of customer

    Example:

    John
  • lastName: (string)

    Last name of customer

    Example:

    Smith
  • address: (string)

    Add whole or part of the address to retrieve matching customers. This could be suburb, postcode or state.

    Example:

    Wyong, Mountain%20Street, 2145
  • pageNo: (number - default: 0)

    The page number to fetch

    Example:

    5
  • pageSize: (number - default: 100)

    The maximum number of records to fetch per page

    Example:

    5
